# Instruções para Hostinger (Hospedagem Compartilhada)

Esta pasta contém os arquivos estáticos necessários para rodar sua landing page na Hostinger.

1. Acesse o **Gerenciador de Arquivos** no seu painel da Hostinger.
2. Vá para a pasta `public_html`.
3. Faça upload de todos os arquivos dentro da pasta `dist_hostinger` diretamente para `public_html`.
4. O arquivo `index.html` deve estar na raiz da `public_html`.

**Nota:** Como se trata de uma landing page estática (Vite), não é necessário configurar Node.js no servidor se você estiver apenas servindo os arquivos estáticos.
